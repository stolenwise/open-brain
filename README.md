# Open Brain — Portfolio Page

A portfolio page documenting the **Open Brain** project: a personal AI memory system built with Slack, Supabase, pgvector, a custom MCP server, and Claude.

## What it is

Open Brain solves the problem of AI memory silos. Every time you open a new chat, your AI starts from zero. This system gives any MCP-compatible AI client persistent, semantically searchable memory — for about $0.20/month.

**Stack:** Slack → Supabase (Postgres + pgvector) → MCP Server → Claude
